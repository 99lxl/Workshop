# YLab-CV-Workshop

## Notice

1. 每周汇报需要准备的内容为 **Note** 和 **Slide**
2. 文件格式：
   - Slide：`PPTX`
   - Note：`MD`
3. 命名规则： 日期-姓名-文件名

	- 日期、姓名、文件名之间以 `-` 连接
	- 其他以 `_` 连接
	- 不要有空格
	- 例如：`2021_10_15-JieWei-Domain_Adaptation`
4. 文件位置：

	- Slide 位置：`Assets/Slides/`
	- Note 位置：`Assets/Notes/`
	- Note 中如果有图片则放入：`Assets/Notes/imgs`
	- 其他文件放入：`Assets/Files`
